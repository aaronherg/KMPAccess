<div align="center">
  <p><strong>🚀 FUNCIONALIDADES PRINCIPALES DE KMPAccess</strong></p>
</div>

<div align="start">

<div align="center"><p><strong>1. Gestión de permisos multiplataforma</strong></p></div>
<ul>
  <li>Solicita y maneja permisos comunes en Android e iOS.</li>
  <li>Gestión de estados: permitido, denegado, restringido.</li>
  <li>Lógica compartida en Kotlin Multiplatform para evitar duplicación de código.</li>
</ul>

<div align="center"><p><strong>2. Ejemplos interactivos</strong></p></div>
<ul>
  <li>Pantallas de prueba para visualizar el estado de los permisos en tiempo real.</li>
  <li>Flujos de permisos explicados paso a paso para aprendizaje.</li>
</ul>

<div align="center"><p><strong>3. Buenas prácticas y arquitectura</strong></p></div>
<ul>
  <li>Uso de coroutines para llamadas asíncronas.</li>
  <li>Separación clara entre lógica compartida y UI específica de cada plataforma.</li>
  <li>Proyecto modular listo para integrar en otros proyectos KMP.</li>
</ul>
